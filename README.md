# 吾輩は猫舌 / wagahai-nekojita

I build small, local-first tools and reviewable AI-assisted development workflows.

My work focuses on turning fast AI-assisted prototyping into something a human can inspect, validate, and safely continue: clear boundaries, reproducible checks, review evidence, and explicit human decisions.

## Representative work

### Forkclip

A local-first macOS clipboard manager built with SwiftUI, AppKit, SQLite, and Keychain.

It represents my product-side work: native Mac UX, private-by-default storage, local persistence, and clear distribution boundaries.

### wn-setup

A personal Codex setup for subagent-first automation, syncable configuration, and machine-verifiable run evidence.

It represents my workflow-side work: permission boundaries, validation scripts, reviewable outputs, and AI-assisted development that does not rely on blind trust.

### GitHub Activity Dashboard

A static demo that turns Git history into reviewable activity narratives.

It represents my interest in making development work easier to inspect after the fact.

### docvault

A Git-backed inbox-zero vault skeleton for Markdown notes.

It represents my approach to local, durable, agent-friendly knowledge workflows.
